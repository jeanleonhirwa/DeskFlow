# Desk Flow - Keyboard Shortcuts Reference

## Navigation Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + 1` | Switch to **Projects** tab |
| `Ctrl + 2` | Switch to **Tasks** tab |
| `Ctrl + 3` | Switch to **Daily Planner** tab |
| `Ctrl + 4` | Switch to **Analytics** tab |

## Action Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + N` | Create **New** item (context-aware: project in Projects view, task in Tasks view) |
| `Ctrl + F` | **Focus** search bar in current view |

## Utility Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + ,` | Open **Settings** dialog |
| `Ctrl + T` | **Toggle** theme (light/dark) |
| `Ctrl + Q` | **Quit** application |

## Dialog Shortcuts

| Shortcut | Action |
|----------|--------|
| `Escape` | Close current dialog |
| `Ctrl + S` | Save (in task/project forms) |
| `Enter` | Save (when not in text area) |

## Tips

- **Context-Aware Actions**: `Ctrl + N` creates different items depending on which view you're in
- **Quick Navigation**: Use number keys (1-4) with Ctrl to jump between views
- **Search Anywhere**: Press `Ctrl + F` to instantly start searching in any view
- **Fast Theme Switch**: Quickly toggle between light and dark modes with `Ctrl + T`

## Power User Workflow Example

1. Press `Ctrl + 2` → Jump to Tasks view
2. Press `Ctrl + F` → Focus search
3. Type your search query
4. Press `Ctrl + N` → Create new task
5. Fill form and press `Ctrl + S` → Save
6. Press `Escape` → Close dialog

All without touching the mouse! ⚡
